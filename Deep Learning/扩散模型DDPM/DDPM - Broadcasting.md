# 概念
广播 = 两个形状不同的张量做运算时，小的那个自动"拉伸"来匹配大的形状。不需要手动复制数据。

用最简单的例子：一个数加到一个矩阵上，写 `矩阵 + 5` 就够了，PyTorch 自动帮你把 5 "复制"到每个格子里。
标量:   5      形状: ()
矩阵: 
       │ 1  │ 2  │ 3  │          
       │ 4  │ 5  │ 6  │
       │ 7  │ 8  │ 9  │

计算 矩阵 + 5 时:
广播 = 把 5 先"脑补"成一个同样大的矩阵：
│ 5  │ 5  │ 5  │       │ 1  │ 2  │ 3  │      │ 6   │ 7   │ 8   │
│ 5  │ 5  │ 5  │  +  │ 4  │ 5  │ 6  │  =  │ 9   │ 10 │ 11 │
│ 5  │ 5  │ 5  │      │ 7   │ 8  │ 9  │      │ 12 │ 13 │ 14 │
