# DDPM：为什么CNN在这里“不够用”？

为什么DDPM需要引入自注意力机制了：
- **浅层网络**：感受野[[CNN知识点#13. 感受野|🔗]]很小，只能看到图像的边缘、颜色斑点、纹理等**最局部的细节**。
- **深层网络**：虽然理论上感受野能覆盖整张图，但它的视野是“强制放大”的，且偏向于关注空间上临近的区域。
这就导致了一个致命问题：生成鸟的左右翅膀时。  
	CNN的深层网络虽然“理论上”能看到左右翅膀，但它的计算逻辑天生就是按空间距离优先的。它很难精确地让“左边的翅膀像素”去主动查找并匹配“右边的翅膀像素”。
	这种跨越长距离、像素级别的“精准配对”，CNN做得很笨拙。
	而**自注意力机制**不讲“距离远近”，它一上来就算法所有像素点之间的相似度，**直接将“感受野”扩展到“全局（整张图）”**。无论左右翅膀隔得多远，它都能瞬间建立联系，保证生成的东西是对称、协调的。
